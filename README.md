# 🎨 AI Clipart Generator

A full-stack mobile application that allows users to upload images and generate AI-style outputs.

---

## 🚀 Features

- 📸 Capture image using Camera  
- 🖼️ Select image from Gallery  
- 👀 Preview selected image  
- 🔗 Send image to backend server  
- 🎨 Generate AI-style output  
- 🖼️ Display generated image  
- 📥 Download / Share generated image  

---

## 🛠️ Tech Stack

### 📱 Frontend
- React Native (Expo)
- TypeScript
- Expo Image Picker
- Expo File System & Sharing

### 🌐 Backend
- Node.js
- Express.js
- Multer (for file uploads)
